## Features

* User management.
* Test cases.
* Test campaigns.
* Test executions.
* Send by e-mail a failing test execution.
* Export to Excel in a click.
* Calendars showing start and end dates of a test campaign.
* Available in English and French.
* Simplified REST API with a .Net client.
